O projeto web foi realizado usando a linguagem PHP.
O projeto tem o intuito de realizar uma pesquisa entre os estudantes de uma determinada faculdade para termos uma ideia de quantos alunos da área da tecnologia têm acesso simples a determinadas coisas.
As perguntas são socioeconômicas e foram desenvolvidas para identificar o perfil dos estudantes, suas condições de estudo, recursos disponíveis e possíveis dificuldades enfrentadas no dia a dia acadêmico.
A partir das respostas coletadas, será possível analisar de forma mais clara a realidade dos alunos e entender quais fatores podem influenciar o seu desempenho e participação no curso. Essas informações 
contribuirão para que a instituição possa criar estratégias de apoio, melhorar políticas internas e oferecer recursos que atendam melhor às necessidades dos estudantes.
O objetivo final é garantir uma visão mais completa sobre a situação socioeconômica dos universitários e, assim, promover ações que ajudem a reduzir desigualdades e aumentar a inclusão dentro do ambiente acadêmico
